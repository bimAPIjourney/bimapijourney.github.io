Revizto is a great software but I miss the ability to batch generate appearance template rules for a property with multiple values.

In Revizto, create a template with the condition you want to replicate:
<img width="437" height="894" alt="image" src="https://github.com/user-attachments/assets/324b7c94-69d2-4292-bf51-e72a63b570ca" />

Navigate to [aptmplwebprocessor.onrender.com](https://aptmplwebprocessor.onrender.com/)

Load the file in the generator and input the property values separated by a comma:
<img width="3520" height="1497" alt="image" src="https://github.com/user-attachments/assets/f2f10f82-2370-45de-90ef-0780a0b8efc0" />

Click on Generate and load the .aptmpl file in Revizto.

Disclaimer:
Tested with Revizto Version 5.16.7
Use at your own risk.
